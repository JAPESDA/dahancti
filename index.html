Sambungkan Google Sheets

Dashboard ini membaca data langsung dari Google Sheets Anda. Lakukan sekali, dan selanjutnya otomatis.

Cara mendapatkan URL:
1. Buka Google Sheets (sudah import Template_DaHan_PCI_v2.xlsx)
2. Klik File → Bagikan → Publikasikan ke web
3. Pilih sheet: tbl_dahan | Format: Nilai dipisah koma (.csv)
4. Klik Publikasikan → salin URL yang muncul
5. Paste di bawah → Sambungkan

URL disimpan di browser Anda — tidak dikirim ke server lain.
